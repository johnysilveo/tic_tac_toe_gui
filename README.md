# Tic Tac Toe GUI

A modern Tic Tac Toe desktop game developed with Python and PySide6.

The project will feature a polished frozen-glass interface with rounded
corners, translucent panels, shadows, reflections, textures, and animations.

## Planned Features

- 3x3 and 5x5 game boards
- Human vs Human mode
- Human vs Computer mode
- Computer vs Computer mode
- Easy, Medium, and Hard AI difficulty levels
- Advanced Minimax AI for the 5x5 board
- Configurable AI search depth
- Configurable AI thinking time
- Background AI calculations without freezing the interface
- Alpha-beta pruning
- Iterative deepening
- Move ordering
- Transposition table
- Frozen-glass visual design
- Rounded controls and panels
- Shadows, reflections, textures, and animations
- Git and GitHub workflow practice

## Technologies

- Python
- PySide6
- Git
- GitHub

This project helps practice Git and GitHub workflows while developing a modern Tic Tac Toe GUI application.
